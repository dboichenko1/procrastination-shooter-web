# Сторож — прокрастинатор-шутер

Стоический «сторож самого себя». Поймал себя на прокрастинации — жмёшь **«Я
прокрастинирую»**, и **Сизиф** берётся за камень и катит его в гору. Катит
непрерывно, пока ты не нажмёшь **«Вернулся к делу»**. Камень всё равно срывается с
вершины вниз — вечный бессмысленный труд как зеркало твоей прокрастинации; на срыве
приходит напоминание «Дядь, точно пора идти дела делать». Раз в час — колокол:
«А не прокрастинируешь ли ты, братец?».

Чистая статика (HTML + CSS + JS), **без сборки, без зависимостей, без сети и без
аналитики**. Ставится на домашний экран iPhone/Android как обычное приложение и
работает полностью офлайн.

**Что внутри:**
- Старт/стоп: жмёшь, когда заметил прокрастинацию, и снова — когда вернулся к делу.
- Сцена Сизифа: камень ползёт в гору и срывается вниз — снова и снова.
- **Статистика**: когда, что и сколько ты прокрастинировал — суммарно за день,
  график по дням, журнал сессий, рекорд, «вернулся к делу N× сегодня».
- Почасовой колокол и напоминание о срыве (по желанию — через уведомления).
- Стоические мысли (Марк Аврелий, Сенека, Эпиктет) и Камю про Сизифа.

## Приватность

Все данные — только на устройстве (`localStorage`). Нет аккаунта, сервера, облака и
сетевых запросов. Экспорт/импорт — локальный JSON-файл. Уведомления планируются
локально, без push-сервера. *«Мы тебе не надзиратель. Ты сам себе надзиратель.»*

## Запуск на GitHub Pages

1. Положите содержимое этой папки в корень репозитория.
2. **Settings → Pages → Source: Deploy from a branch**, ветка `main`, папка `/ (root)`.
3. Через минуту приложение откроется по адресу
   `https://<логин>.github.io/<репозиторий>/`.

Все пути относительные; `.nojekyll` отключает Jekyll.

## Установка на телефон

- **iPhone (Safari):** «Поделиться» → **«На экран „Домой"»**.
- **Android (Chrome):** меню → **«Установить приложение»**.

Разреши уведомления в настройках (⚙), чтобы получать колокол и напоминания.

## Управление

- Большая кнопка — **старт/стоп** (Пробел/Enter с клавиатуры).
- **Esc** — мгновенно остановить (вернуться к делу).
- **▦** — статистика, **⚙** — настройки (звук, колокол, тихие часы, наставник, данные).
